# Public Assets Directory

This directory contains static assets that are served directly by the application.

## Files in this directory:

### templates/Cmr.xls
The Excel template used for CMR (Convention relative au contrat de transport international de marchandises par route) document generation.

**Status:** ✅ **READY TO USE**

The file is located at `/workspaces/spark-template/public/templates/Cmr.xls` and is configured as the default CMR template.

This template is used by the CMR export functionality in the Orders tab. The application references it from `/templates/Cmr.xls`.

### templates/
Directory containing additional export templates and documentation.
